# Password Strength Checker

A GUI-based Python application that evaluates the strength of a password based on common security criteria and provides suggestions to create a stronger password.

## Features
- Check password length
- Detect uppercase and lowercase letters
- Check for numbers
- Check for special characters
- Display password strength (Weak, Medium, Strong)
- Provide suggestions to improve password security
- User-friendly GUI using Tkinter

## Technologies Used
- Python
- Tkinter
- Regular Expressions (re)

## How to Run
1. Run the Python program.
2. Enter a password.
3. Click the "Analyze Password" button.
4. View the password strength and suggestions.

## Learning Outcomes
- GUI development with Tkinter
- Password validation techniques
- Regular Expressions (Regex)
- User input handling
- Python programming
